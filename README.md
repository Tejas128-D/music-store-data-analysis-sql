# 🎵 Music Store Data Analysis | SQL Project

## Author

Tejas R

## Project Overview

In this project, I analyzed a music store database using SQL to understand customer behavior, sales performance, and music trends.
The goal of the project is to answer business questions and generate useful insights from the data.

## Database and Tools Used

* PostgreSQL
* PgAdmin4
* SQL

## Dataset Description

The dataset represents a digital music store and includes tables such as:

* Customers
* Employees
* Artists
* Albums
* Tracks
* Genres
* Invoices
* Invoice Lines

These tables are connected using primary and foreign keys.

## Business Questions Solved

Some of the business questions answered in this project include:

* Who is the senior most employee based on job title?
* Which countries have the most invoices?
* What are the top 3 invoice values?
* Which city generated the highest revenue?
* Who is the best customer?
* Which artists have written the most rock music?
* Which tracks are longer than the average song length?
* Which genre is most popular in each country?
* Which customer spent the most money in each country?

## SQL Concepts Used

This project helped me practice different SQL concepts such as:

* SELECT statements
* JOIN operations
* GROUP BY and ORDER BY
* Aggregation functions (SUM, COUNT, AVG)
* Subqueries
* Common Table Expressions (CTE)
* Window functions (ROW_NUMBER)
